Compétences

HTML (HyperText Markup Language) est le langage standard utilisé pour structurer le contenu des pages web incluant textes, images et liens.
CSS (Cascading Style Sheets) est utilisé pour styliser et mettre en forme le contenu des pages web, incluant couleurs, polices et mises en page.
JavaScript est un langage de programmation côté client qui permet de rendre les pages web interactives, dynamiques, et réactives.
Java est un langage de programmation orienté objet, largement utilisé pour le développement d’applications mobiles, serveurs, et logiciels d'entreprise.
MySQL est un système de gestion de bases de données relationnelles open-source, utilisé pour stocker et gérer des données sur des sites web.
Git est un système de contrôle de version distribué qui permet de suivre les modifications du code source et de collaborer efficacement en équipe.
Python est un langage de programmation polyvalent, connu pour sa simplicité et sa lisibilité, utilisé en développement web, analyse de données, et IA.
PHP est un langage de script côté serveur utilisé pour le développement web dynamique, souvent pour créer des sites et applications interactifs.
